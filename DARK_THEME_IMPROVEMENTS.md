# Улучшения темной темы Dashboard

## Изменения

Обновлен файл [src/main/resources/io/jenkins/plugins/swarmcloud/SwarmDashboard/index.jelly](src/main/resources/io/jenkins/plugins/swarmcloud/SwarmDashboard/index.jelly) для поддержки темной темы Jenkins.

### Что было сделано:

1. **Добавлены CSS переменные для цветовой схемы:**
   - `--swarm-bg` - фон страницы
   - `--swarm-card-bg` - фон карточек
   - `--swarm-border` - цвет границ
   - `--swarm-input-bg` - фон элементов ввода/статистики
   - `--swarm-text` - основной цвет текста
   - `--swarm-text-secondary` - вторичный цвет текста
   - `--swarm-hover-bg` - фон при наведении

2. **Поддержка темной темы:**
   - Через `@media (prefers-color-scheme: dark)` - автоопределение системной темы
   - Через `[data-theme="dark"]` - поддержка темной темы Jenkins

3. **Обновлены компоненты:**
   - ✅ Карточки (cards) - теперь используют динамические цвета
   - ✅ Статистика (stats) - адаптирована под обе темы
   - ✅ Таблицы - границы и текст правильно отображаются
   - ✅ Badges - используют контрастные цвета, читаемые в обеих темах:
     - Success (зеленый): `hsl(142 76% 36%)` с белым текстом
     - Warning (оранжевый): `hsl(32 95% 44%)` с белым текстом
     - Danger (красный): `hsl(0 84% 48%)` с белым текстом
     - Info (синий): `hsl(199 89% 40%)` с белым текстом
     - Secondary: использует динамические переменные
   - ✅ Кнопки - инвертируют цвета в зависимости от темы
   - ✅ Сообщения об ошибках - темный фон с светлым текстом
   - ✅ Ссылки - используют более яркий синий для читаемости

## Цветовая палитра

### Светлая тема (по умолчанию):
```css
--swarm-bg: #ffffff;
--swarm-card-bg: #ffffff;
--swarm-border: hsl(214 32% 91%);
--swarm-input-bg: hsl(210 40% 98%);
--swarm-text: hsl(222 47% 11%);
--swarm-text-secondary: hsl(215 16% 47%);
--swarm-hover-bg: hsl(210 40% 98%);
```

### Темная тема:
```css
--swarm-bg: hsl(222 47% 11%);          /* Темно-синий фон */
--swarm-card-bg: hsl(217 33% 17%);    /* Более светлый фон для карточек */
--swarm-border: hsl(217 33% 25%);     /* Границы */
--swarm-input-bg: hsl(217 33% 20%);   /* Фон элементов */
--swarm-text: hsl(210 40% 98%);       /* Светлый текст */
--swarm-text-secondary: hsl(215 20% 65%); /* Вторичный текст */
--swarm-hover-bg: hsl(217 33% 22%);   /* Фон при hover */
```

## Преимущества:

1. **Автоматическое определение** - если Jenkins использует темную тему, Dashboard автоматически адаптируется
2. **Системная интеграция** - поддержка системной темы ОС через `prefers-color-scheme`
3. **Контрастность** - все элементы читаемы в обеих темах
4. **Консистентность** - единый стиль с shadcn/ui
5. **Производительность** - используются CSS переменные без дополнительного JavaScript

## Использование:

После установки обновленного плагина Dashboard будет автоматически переключаться между светлой и темной темой в зависимости от настроек Jenkins или системы.

### Проверка:

1. Откройте Dashboard: `http://jenkins/swarm-dashboard/`
2. Переключите тему Jenkins (если поддерживается)
3. Все элементы должны корректно отображаться

### Браузерная поддержка:

- Chrome/Edge 76+
- Firefox 67+
- Safari 12.1+

## Скриншоты

### Светлая тема:
- Четкие границы
- Высокий контраст текста
- Яркие badges

### Темная тема:
- Темный фон без бликов
- Комфортный для глаз текст
- Контрастные badges остаются читаемыми

## Обратная совместимость:

Все изменения обратно совместимы. Если браузер не поддерживает CSS переменные или темную тему, будет использована светлая тема по умолчанию.
